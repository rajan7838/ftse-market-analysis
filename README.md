# FTSE Market Analysis - Unsupervised ML Project

## Overview
This project uses unsupervised machine learning to analyze FTSE stock market data, identifying patterns, clusters, and anomalies in market behavior.

## Features
- **PCA** - Dimensionality reduction for visualization
- **K-Means Clustering** - Group similar trading days
- **Hierarchical Clustering** - Tree-based clustering
- **DBSCAN** - Density-based clustering with outlier detection
- **Isolation Forest** - Anomaly detection for unusual market days
- **Streamlit App** - Interactive web interface
